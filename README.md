# Customer Churn Prediction Project

## Overview
This project identifies credit card customers likely to churn using machine learning. 

## Project Structure
.
├── data
│ └── bank_data.csv
├── images
│ ├── eda
│ └── results
├── logs
├── models
├── churn_library.py
├── churn_script_logging_and_tests.py
├── churn_notebook.ipynb
└── README.md

## Features

- Exploratory Data Analysis (EDA)
- Feature engineering with categorical encoding
- Model training (Random Forest, Logistic Regression)
- Model evaluation (classification reports, ROC)
- Feature importance visualization
- Logging and testing
